Health Care — Système de Gestion Médicale

Description:
Health Care est une application de gestion médicale qui permet de gérer les patients, les médecins, les dossiers médicaux et les rendez-vous. 
Elle expose une API sécurisée par authentification JWT, consommable par n'importe quel client front-end ou mobile.

Fonctionnalités:
Authentification — Inscription et connexion avec génération de token JWT
Gestion des patients — Création, consultation, modification et suppression
Gestion des médecins — CRUD complet avec spécialités
Dossiers médicaux — Création et consultation des dossiers liés aux patients
Rendez-vous — Planification et suivi des rendez-vous (statuts : PLANIFIÉ, CONFIRMÉ, ANNULÉ)
Documentation API — Interface Swagger UI intégrée
